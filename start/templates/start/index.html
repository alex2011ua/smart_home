Живу в частном доме. В свободное время занимаюсь реализацией и постоянной модернизацией личного проекта "Умный дом".

Реализовал проект умного дома на Django.
Запущен сервер на мини компьютере Raspberry на котором установил и настроил Nginx, Gunicorn.
К серверу подключен контроллер Arduino для работы с датчиками.
При помощи Celery настроил регулярный опрос датчиков и выполнение длительный операций.
Информация сохраняется в SQLite

Реализованный функционал:
  • По API раз в день запрашивается прогноз погоды для автоматического полива.
  • Зимой реализовано удаленное управление уличной иллюминацией.
  • Подключены и реализована запись в БД и вывод на сайт значений различных датчиков, а именно: температура, открытия двери, качества воздуха, и др.
  • Реализовано удаленное управление различными устройствами.
  • Подключил Телеграмм и Вайбер боты для получения важной информации.
  • Идентификация и аутентификация пользователя встроенными средствами Django.
  • Подключена админка.
